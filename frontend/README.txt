Discover Saudi Arabia – Interactive Cultural Website
====================================================

This is a unique HTML/CSS/JavaScript frontend version of the project.

Project name kept as:
Discover Saudi Arabia – Interactive Cultural Website

Design concept:
- Arabic cultural travel passport
- Interactive region cards
- Gallery search and filtering
- Dynamic details page using localStorage
- Admin dashboard with CRUD simulation using localStorage
- Night mode
- Arabic RTL interface

How to run:
1. Extract the ZIP file.
2. Open index.html in your browser.
3. For a better experience, open the folder in VS Code and use Live Server.
